Чаты для клиентов

Модуль отдает публичный скрипт для Tilda. Скрипт читает параметры email и phone из адресной строки, вызывает /tilda-chat-links/api/lookup, проверяет GetCourse и показывает ссылки на актуальные TG/VK чаты.

Нужные ENV: GETCOURSE_ACCOUNT_NAME, GETCOURSE_API_TOKEN. Сначала используются допполя GetCourse Ссылка на чат ВК и Ссылка на чат ТГ, затем группы пользователя, затем актуальные ссылки из Google Sheet.

Частично оплаченный клиент получает ссылки, пока возраст частичной оплаты не превышает TILDA_CHAT_LINKS_PARTIAL_GRACE_DAYS дней. После этого виджет показывает «Ссылки недоступны».

В панели модуля есть бан-лист по email или телефону. Бан-лист проверяется до кэша и сразу перекрывает выдачу ссылок.

Тяжелый скан заказов GetCourse выключен по умолчанию: TILDA_CHAT_LINKS_ENABLE_DEAL_SCAN=false. Для снижения расхода лимитов включен SQLite-кэш проверок клиента.
