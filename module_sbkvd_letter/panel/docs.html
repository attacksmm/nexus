Модуль «Рассылки»

Отправляет сообщения VK и Telegram по сохранённым сегментам из разрешённых листов «Базы клиентов» и ручных получателей.

Настройка

  1. На вкладке «Настройки» добавьте VK и Telegram bot token. Значения сохраняются в ENV Nexus и не возвращаются в браузер.
  2. Разрешите листы, выберите канал и поле с ID получателя.
  3. На вкладке «Сегменты» создайте аудиторию по последовательным блокам: источники, условия, корректировки и итог.
  4. На вкладке «Шаблоны» загрузите вложения и соберите общую VK/Telegram-клавиатуру из рядов и кнопок. Кнопки можно переносить drag-and-drop; тот же state доступен в универсальном JSON.
  5. На вкладке «Рассылка» выберите готовый сегмент между названием и текстом сообщения.

Хранение

Шаблоны и вложения находятся в системной папке sbkvd-letter Файлового хранилища. Кампании, snapshot сегмента, получатели, попытки и внешние ID сообщений находятся в SQLite модуля.

Надёжность

При запуске backend загружает выбранный сегмент по ID и фиксирует его snapshot. Изменение сегмента не меняет уже созданную рассылку. Универсальная клавиатура компилируется в нативные форматы VK и Telegram непосредственно перед отправкой.

Права

Viewer просматривает данные, editor управляет шаблонами, сегментами и рассылками, admin дополнительно настраивает листы, токены и удаляет историю.
