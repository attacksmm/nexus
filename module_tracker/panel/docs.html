Трекер

Модуль собирает визиты, UTM, click id, IP/Geo, устройство и подтвержденные формы. Скрипт не меняет страницу: не правит URL, DOM, стили, формы и не блокирует отправку.

Установка скрипта

<script src="https://junior.sobakovod.pro/nexus/tracker/api/script.js" async data-site="sobakovod" data-consent="required" data-banner="1" data-policy-url="https://sobakovod.pro/popd" data-auto-consent="continue"></script>

data-site нужен только для группировки сайтов в панели. Если не задан, используется host страницы.

Согласие и баннер

data-consent="required" включает режим, при котором скрипт до согласия не ставит visitor cookie/localStorage и не отправляет визиты/формы. data-banner="1" показывает стандартный нижний cookie-баннер в стиле сайта: читаемый текст, ссылка на политику и кнопка принятия. data-auto-consent="continue" считает согласием продолжение пользования после небольшой задержки: скролл, клик по сайту, касание или клавишу. Для более строгого режима уберите data-auto-consent и принимайте согласие только кнопкой баннера или вызовом API.

window.NexusTracker.grantConsent("site_banner");
window.NexusTracker.revokeConsent();
window.NexusTracker.hasConsent();

Что собирается

ГруппаПоля
ВизитURL, referrer, title, host, UTM, yclid/gclid/fbclid/ttclid/msclkid/roistat/_ym_uid, browser context.
Идентификацияvisitor_id, visit_id, email, phone, name, utm_term, fingerprint, marketing ids.
СерверIP, User-Agent, browser/device, GeoIP country/city при наличии GeoLite2-City.mmdb.
ФормыТолько подтвержденные формы. Пароли, токены, hidden, file, captcha/OTP/CSRF и карточные поля исключаются.

Подтверждение формы

Автоматически срабатывают success URL и видимые success-блоки Tilda/GetCourse-подобных форм. Для SPA можно вызвать:

window.NexusTracker.confirmForm(document.querySelector("form"), { confirmation_reason: "crm_success" });

Хранение

Raw-события чистятся вручную во вкладке «Настройки» по retention, по умолчанию 6 месяцев. Склеенные профили людей остаются в БД модуля и синхронизируются в customer-db таблицу visitor_profiles.
