Создание учебных чатов

Назначение

Модуль переносит legacy endpoint'ы Sobakovod/Windsurf для создания учебных чатов в Nexus и добавляет панель управления списками людей, курсами и шаблонами сообщений.

Публичные endpoint'ы

POST /nexus/course-chat-creator/api/vk/create
POST /nexus/course-chat-creator/api/telegram/create
POST /nexus/course-chat-creator/api/chats/create

Для старых сценариев оставлены алиасы /process_vk и /process6. Публичные webhook-вызовы требуют поле password, если задан NEXUS_CHAT_CREATOR_PASSWORD. Запуск из панели Nexus работает по авторизации Nexus. Документация открывается штатной кнопкой Nexus ? у модуля.

Поля запроса

{
  "password": "...",
  "stream_number": "51",
  "start_date": "01.06.2026",
  "date_start": "01.06.2026",
  "course_choice": "1",
  "course_type": "puppy",
  "test_mode": "true"
}
  • course_choice=1 и course_type=puppy означают курс щенка.
  • course_choice=2 и course_type=dog означают основной курс собаки.
  • Кураторы выбираются по четности stream_number: even, odd или any.
  • test_mode=true создает пустой чат без приглашения людей, бота и назначения админов.

ENV

Минимально нужны только ключи ниже. Telegram session создаётся во вкладке Telegram внутри панели. Webhook password нужен только для внешних публичных вызовов.

VK_USER_TOKEN=
TELEGRAM_API_ID=
TELEGRAM_API_HASH=

Опционально: NEXUS_CHAT_CREATOR_PASSWORD или SBKVD_PROCESS_WEBHOOK_PASSWORD для webhook, VK_GROUP_TOKEN/VK_GROUP_ID для сообщества, VK_TEST_USER_TOKEN для отдельного test token, TELEGRAM_SESSION_FILE только если нужен нестандартный путь.

Что сохраняется

  • VK: создание беседы, добавление сотрудников, назначение всем сотрудникам роли admin, аватар из static/group_photo.jpg, история для новых участников, приветствие, закреп, invite link.
  • Telegram: создание forum supergroup, обязательная проверка тем Инфо/Визитка/Обучение/Болталка, аватар из static/group_photo.jpg, бот, полные права администратора всем сотрудникам без anonymous, приветственные сообщения, закрепы, invite link.
  • Панель: редактирование и удаление людей, курсов, шаблонов, Telegram-авторизация, удаление людей только из учебных чатов и просмотр истории запусков.
  • Шаблоны по умолчанию взяты из Windsurf/Sobakovod endpoint'ов.

Люди

Для сотрудника указываются имя, роль, VK страница/username/ID и Telegram username. В текстах приветствия используются username, не телефоны. Текущий базовый состав: Анна — автор курса, Ирина — куратор, остальные сотрудники — админы без отдельной видимой подписи администратора.

Удаление

Вкладка удаления сначала показывает dry-run. Реальное удаление применяется только к чатам с названием потока вида 51. 01.06.2026 - ... и курсом Собаковода.
