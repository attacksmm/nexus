Сохранённые кнопки

Заполните данные из Senler. После сохранения модуль выдаст автономный блок: официальный Senler + встроенный адаптер для Salebot/Tilda без загрузки скрипта с нашего сервера.

Автономная прямая ссылка без загрузки с нашего сервера. Если Salebot отрисовал VK-кнопку, скрипт заменит её. Если Salebot-блок пустой, поставит одну кнопку сам.


      
Сохранение работает для пресета слева: название, group/subscription, текст, selector и redirect. Вкладки строят скрипт из выбранного пресета; чтобы добавить новую кнопку, нажмите “+ Новая”, заполните поля и сохраните. Чтобы править старую, выберите её слева, измените поля и сохраните. “Удалить” удаляет выбранный пресет.
