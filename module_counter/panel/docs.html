Подсчет v1.0.0

Модуль читает листы customer-db, считает строки по выбранным условиям и выгружает найденных клиентов в CSV.

Логика

  • Подсчет идет по строкам выбранных листов. Один человек в VK и Telegram считается двумя строками.
  • Поддерживаются системные поля id, platform_id, created_at, updated_at и JSON-пути из custom_fields.
  • Примеры JSON-путей: utms.ym_uid, utms.yclid, contact_fields.phone, possible_accounts.salebot_id.
  • Модуль только читает customer-db и не изменяет клиентские записи.

Операторы

равно, не равно, содержит, не содержит, пусто, не пусто, начинается с, заканчивается на, больше, меньше, между, in list.
