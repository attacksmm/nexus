Файловое хранилище

Модуль хранит файлы в данных модуля Nexus, позволяет создавать папки, текстовые файлы и загружать разрешённые безопасные типы.

Прямые ссылки

У каждого файла есть публичная ссылка вида /file-storage/api/f/{token}/{filename}. Она открывается без входа в Nexus, но содержит длинный случайный токен.

Разрешённые типы

Разрешены изображения, видео, аудио, txt, md, csv, json, jsonl, log, pdf, docx, xlsx, zip. Исполняемые, HTML, JS и SVG запрещены.

Ограничения

  • Один файл: до 100 MB.
  • Редактирование текста в панели: до 1 MB.
  • Системные пути не используют пользовательские имена файлов.
  • Удаление папки удаляет все вложенные файлы и публичные ссылки.
