Файловое хранилище

Модуль хранит файлы в данных модуля Nexus, позволяет создавать папки, текстовые файлы и загружать файлы любого типа.

Прямые ссылки

У каждого файла есть прямая ссылка вида /file-storage/api/f/{token}/{filename}. По умолчанию она публичная и защищена длинным случайным токеном. В окне файла можно включить режим «Только авторизованным»: тогда скачивание по этой же ссылке требует действующую cookie входа в Nexus и доступ пользователя к модулю.

Загрузка

Тип и расширение файла не ограничиваются приложением. Большие файлы загружаются чанками и могут продолжаться после обновления страницы, если браузер сохранил доступ к выбранному локальному файлу. Если доступ потерян, выберите тот же файл снова.

Ограничения

  • Лимит размера в приложении: не задан. Фактический предел определяется свободным местом на диске, nginx/proxy и стабильностью соединения.
  • Редактирование текста в панели: до 1 MB.
  • Системные пути не используют пользовательские имена файлов.
  • Потенциально исполняемые в браузере типы по публичной ссылке отдаются как скачивание, а не inline-просмотр.
  • Для ссылок с режимом «Только авторизованным» сервер отдает файл с приватным cache-control.
  • Удаление папки удаляет все вложенные файлы и публичные ссылки.
