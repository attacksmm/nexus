Списки Сенлер v1.0.0

Модуль автоматически добавляет VK-пользователей в списки рассылки Senler при посещении ими страниц со встроенным JS-трекером.

Принцип работы

  1. На сайт вставляется JS-скрипт (вкладка «Скрипт»)
  2. При каждом посещении скрипт отправляет POST /nexus/senler/api/track с URL и параметрами страницы
  3. Модуль регистрирует страницу и ищет активные связки для этого URL
  4. Если связка найдена и в параметрах есть VK ID — вызывается Senler API для добавления в список

REST API

Базовый путь: /nexus/senler/api

МетодПутьОписание
GET/statsОбщая статистика
GET/POST/trackТрекинг-эндпоинт (CORS *)
GET/pagesСписок зарегистрированных страниц
DELETE/pages/{id}Удалить страницу и её связки
GET/bindingsСписок связок
POST/bindings{page_url, subscription_id, vk_id_param, note}
PUT/bindings/{id}/toggleВкл/выкл связку
DELETE/bindings/{id}Удалить связку
GET/visitsЖурнал визитов (limit=200)
GET/settingsНастройки API (без значений токенов)
POST/settings{access_token, group_id}

Трекинг-запрос

POST /nexus/senler/api/track
Content-Type: application/json
{
  "url": "https://example.com/page",
  "params": { "vk_id": "123456", "utm_source": "vk" }
}

Структура связки

ПолеТипОписание
page_urlTEXTURL без параметров (origin+path)
subscription_idTEXTID списка рассылки в Senler
vk_id_paramTEXTИмя параметра URL содержащего VK ID (default: vk_id)
noteTEXTПроизвольная заметка
activeINTEGER1=активна, 0=отключена
